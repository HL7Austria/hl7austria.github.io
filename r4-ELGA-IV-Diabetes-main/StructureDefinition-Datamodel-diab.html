Logical Model: Datamodel Diabetes

Official URL: https://diab.iv.elga.gv.at/StructureDefinition/Datamodel-diab Version: 0.1.0
Draft as of 2024-05-27 Computable Name: DatamodelDiab

Datenmodel das basierend auf den Datenarten für das Versorgungsstufenübergreifende elektronische Leitdokument des Rahmenkonzepts Integrierte Versorgung Diabetes Mellitus Typ 2 erstellt wurde.

Refer to the ​mapping from the logical model of diabetes to the logical model based on the International Patient Summary (IPS) in order to get an idea how the IPS can be used in this context.

Usage:

  • This Logical Model Profile is not used by any profiles in this Implementation Guide
