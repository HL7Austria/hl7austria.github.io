CodeSystem: KIOLA Measurement Codes

Official URL: https://fhir.ehealth-systems.at/kiola/measurement Version: 0.1.0
Active as of 2024-03-11 Computable Name: KIOLAMeasurementCS

Types of vital data measurements that can be transmitted to a KIOLA system. The naming is derived from MDC display names of devices from the ISO/IEEE 11073 standard.

This Code system is referenced in the content logical definition of the following value sets:

  • KIOLAGenericQuestionnaires
  • KIOLAMeasurementTypes
  • KIOLAMedicationDailyDosage
