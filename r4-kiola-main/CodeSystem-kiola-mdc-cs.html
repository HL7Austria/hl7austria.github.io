CodeSystem: KIOLA Device Codes AIT 11073

Official URL: https://fhir.ehealth-systems.at/kiola/device/AIT-11073 Version: 0.1.0
Active as of 2024-03-11 Computable Name: KIOLAMDCCS

Measurement device types for automatic transmission of measurements and their configuration parameters, based on the ISO/IEEE 11073 standard with vendor-specific extensions, mainly AIT. The MDC display name is used as a code.

This Code system is referenced in the content logical definition of the following value sets:

  • KIOLAMeasurementAutomaticTransmissionDeviceTypes
  • KIOLAMeasurementAutomaticTransmissionProperties

Properties

This code system defines the following properties for its concepts

CodeTypeDescription
abstractbooleanTrue, if an element is considered abstract = not for use.
