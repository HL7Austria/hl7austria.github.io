Technical issues

Suggestions for improvements or error messages detected during the use of TerminoloGit can be reported via the GitLab ticket system at https://gitlab.com/elga-gmbh/termgit-dev/-/issues/new.

Issues regarding terminologies

All questions regarding ELGA and Austrian e-Health relevant terminologies that cannot be answered with this documentation can be reported to cda@elga.gv.at.
